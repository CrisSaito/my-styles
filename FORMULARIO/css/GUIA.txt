// GUIA DE ESTILOS PARA O PORTAL DA ADVISEU //

** As classes são baseadas na versão v3.3.7 do BOOTSTRAP (bootstrap.css). 
** É necessário ter uma versão do Bootsrap vinculado no documento HTML.

:: Basicamente, aos nomes das classes do Bootstrap é acrescentado o prefixo "adv-". Em alguns casos, serão necessários os nomes originais das classes do Bootstrap. Exemplo: class="nav adv-nav".


/**************/
/*** BOTÕES ***/
/**************/
:: Há duas versões:
	- NORMAL: traz o botão com preenchimento COLORIDO de acordo com o tipo (primary, danger, etc).
		Exemplo: class="adv-btn adv-btn-primary"

	- INVERTIDO: traz o botão com preenchimento BRANCO, com CONTORNO COLORIDO de acordo com o tipo (primary, danger, etc).
		Exemplo: class="adv-btn adv-btn-primary-invert"
